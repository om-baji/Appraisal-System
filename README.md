# Software_engineering_cp
Requirements are as follows 
- Add, update, delete, and manage employee profiles.
- Define appraisal criteria and performance metrics.
- Allow managers to evaluate employees based on predefined parameters.
- Enable self-appraisal submissions by employees.
- Calculate overall performance ratings automatically.
- Generate appraisal reports and performance summaries.
- Provide secure login access for HR, managers, and employees.
